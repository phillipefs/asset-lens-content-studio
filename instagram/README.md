# Entregas

Cada publicação ocupa `instagram/YYYY-MM-DD-slug/` e contém:

- `render.json` — fonte visual versionável;
- `copy.md` — legenda versionável;
- `post.png` ou `slide-*.png` — artefato final local;
- `render*.html` — artefato intermediário local.

PNGs e HTMLs são ignorados pelo Git, mas continuam obrigatórios na entrega e
devem passar pelo verificador da skill.
