# Screenshots de Entrada

## Regras

- capture a interface real com escala 2x quando possível;
- recorte no elemento relevante, sem margens vazias excessivas;
- preserve textos, números, cores e gráficos originais;
- não inclua credenciais, e-mails, IDs privados ou dados pessoais;
- crie um `.md` adjacente com origem, data, escopo e observações;
- trate toda métrica visível como snapshot mutável.

Nome sugerido: `YYYY-MM-DD-descricao-curta.png`.
