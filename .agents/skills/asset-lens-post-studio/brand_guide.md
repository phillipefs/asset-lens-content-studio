# Brand Guide — Asset Lens

## Posicionamento

O Asset Lens é uma plataforma informacional para organizar métricas
históricas, drawdown, ciclos, risco e leituras quantitativas do mercado
brasileiro.

## Nome oficial

**Asset Lens**

Nunca "Asset Lens Data". Referências antigas em `approved_examples/` ainda
exibem o nome antigo: valem como referência de estilo, nunca de marca.

## Logo

`references/brand/logo_official.png` — badge circular com monograma "AL",
canal alpha obrigatório. Não redesenhar, não aplicar caixa preta, não
substituir por ícone genérico.

## Linguagem

- objetiva, profissional, informacional, acessível;
- sem promessa de retorno, sem indicação de compra/venda, sem tom de
  recomendação (lista completa de termos proibidos no `SKILL.md`).

## Paleta (tokens do renderizador)

| Papel | Cor | Hex |
|---|---|---|
| fundo (topo → base) | navy quase preto | `#020810` → `#010409` |
| superfícies | azul-marinho profundo | `#07111D` / `#0A1828` / `#0E1F31` |
| borda de painéis | azul petróleo | `#125676` |
| destaque primário | ciano | `#22D8F4` |
| destaques de apoio | teal / azul / verde | `#15E8D1` / `#41B4FF` / `#39DB9B` |
| texto principal | branco azulado | `#F4F7FB` |
| texto secundário / terciário | cinza azulado | `#BFCAD8` / `#8FA0B4` |
| indicadores específicos | amarelo / vermelho | `#FFBF57` / `#FF667B` |

Vermelho e amarelo só em indicadores (drawdown, faixas, alertas) — nunca como
cor decorativa.

## Tipografia

- Títulos: **Poppins** (SemiBold/Bold) — embutida em `assets/fonts/`;
- Texto: **Inter** (variável) — embutida em `assets/fonts/`;
- Fallback local: Segoe UI.

## Assinatura visual

- dark premium fintech; contraste alto e controlado;
- glow sutil (nunca neon estourado);
- painéis glass com borda ciana translúcida e cantos arredondados;
- textura de pontos discreta e marca d'água "A" apenas quando melhorar a
  composição;
- molduras refinadas, layouts modernos, tipografia limpa.

## Disclaimer padrão

> Conteúdo informacional. Não é recomendação de investimento.
